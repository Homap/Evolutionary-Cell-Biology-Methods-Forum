# Methods Corner

This section collects short methodological explainers connected to the reading group.

Each method page explains:

1. what the method does
2. what kind of biological question it helps answer
3. how it connects to evolutionary cell biology
4. example use cases
5. optional papers or tools

The goal is not to create a complete methods textbook, but to build a shared vocabulary for discussing modern evolutionary cell biology.
